# 🐢 거북이를 지켜라! 게임

하늘에서 떨어지는 무시무시한 과제들을 피하는 생존 게임입니다!
파이썬 `turtle` 라이브러리를 활용하여 제작했습니다.

## 📅 기획 의도
기말고사 기간, 쏟아지는 과제에 고통받는 대학생의 마음을 담았습니다.
하늘에서 떨어지는 '과제'를 피하며 소소한 스트레스를 해소하고자 기획했습니다.

## 🎮 게임 기능 및 특징
1. **시작 화면 구현:** `START` 버튼을 클릭하여 게임을 시작할 수 있습니다.
2. **부드러운 조작감:** 방향키를 누르고 있을 때만 움직이는 직관적인 컨트롤을 구현했습니다.
3. **난이도 상승:** 시간이 지날수록 과제가 떨어지는 속도가 점점 빨라집니다.
4. **시각적 요소:** 파스텔톤 색감과 가독성 높은 디자인을 적용했습니다.

## 🕹️ 조작 방법
- **마우스 클릭:** 게임 시작 (START 버튼)
- **← (왼쪽 방향키):** 왼쪽으로 이동
- **→ (오른쪽 방향키):** 오른쪽으로 이동
- **R 키:** 게임 오버 시 재시작
